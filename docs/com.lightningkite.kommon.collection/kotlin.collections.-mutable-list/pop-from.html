popFrom

fun <E> MutableList<E>.popFrom(from: E): E?

Pops the from state off the stack. If from is not on the stack, then this will do nothing and return false. If from is the last item on the stack, then this will do nothing and return false. If from is not the last item on the stack, items will be popped off until it is.
